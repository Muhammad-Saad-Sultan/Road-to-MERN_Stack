# Personal Portfolio Website - Muhammad Saad Sultan

Welcome to my personal portfolio website! This project serves as a live developer portfolio showcasing my skills, projects, background, and contact information.

## Project Structure

```text
portfolio/
├── index.html          # Main HTML structure with semantic sections
├── style.css           # External stylesheet for design system & grid layout
├── images/             # Media and icon assets
│   ├── main.png        # Profile headshot
│   └── portfolio-icon.png
└── README.md           # Project overview and documentation
```

## Technical Stack

- **HTML5** (Semantic Elements, Forms, Attributes)
- **CSS3** (Flexbox, CSS Grid, Custom Properties, Media Queries)
- **FontAwesome** (Vector icons)
- **Google Fonts** (Poppins typography)
